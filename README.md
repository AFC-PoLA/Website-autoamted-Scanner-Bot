## Features

- **35+ Security Scans** - Comprehensive vulnerability testing
- **WordPress Detection** - User enumeration, plugins, themes, versions
- **Vulnerability Scanning** - SQL Injection, XSS, LFI/RFI testing
- **Network Tools** - Port scanning, subdomain enumeration, DNS lookup
- **OSINT Tools** - WHOIS lookup, technology detection, WAF detection
- **Admin Panel Finder** - Discover hidden admin pages
- **Load Testing** - Stress test with up to 6000 requests
- **PDF Reports** - Generate professional security reports
- **Free to Use** - No subscription required

## How to Use

1. Open the bot: **[Bot](https://t.me/Fbi_wp_hacker_bot)**
2. Send `/start` to begin
3. Send any website URL
4. Select the scans you want to run
5. Get instant security results!
